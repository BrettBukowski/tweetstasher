# Tweetstasher

Website used to save drafts of tweets and post them to Twitter later.

Node.js app using CouchDB as the datastore and Redis as the session store.

Makes use of many fine libraries and tools:

#### Server

* Express
* LESS
* EveryAuth
* Connect-Redis
* Cradle

#### Browser

* Require.js
* Underscore
* Backbone
* JQuery

#### Testing

* Mocha
* Sinon
* Zombie
* Sandboxed-module
